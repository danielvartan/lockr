## Overview

`lockr` is a user-friendly R package designed for encrypting/decrypting
files.

## Installation

You can install `lockr` using the
[`remotes`](https://github.com/r-lib/remotes) package:

``` r
remotes::install_github("danielvartan/lockr")
```
